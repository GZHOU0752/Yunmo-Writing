# 战斗场景写作指南

## 核心原则：战斗即叙事
战斗本身是故事的一部分，用血与伤疤来书写，不是中断剧情的动作秀。

## 节奏控制
- **快速暴力**：短句、断句、单词独立成段。强迫眼睛快速移动，模拟呼吸急促。
- **震惊/痛苦/顿悟**：长句、接续句。凝固时间，形成慢镜头效果。
- **动作→反应→代价**：每个攻击必须有后果。击中对手后，描写对手的反应；被击中后，描写伤势对后续动作的影响。

## 动词即武器
- 禁止被动语态。不说"他被一拳击中"，说"拳头砸进他的肋骨"。
- 使用具体动词代替抽象词：砸、劈、撕、碾、绞、贯、崩、震。
- 避免"打击""攻击""防御"等概括性词汇。

## 环境即武器
- 战场环境是战斗参与者：浑浊的地面、狭窄的走廊、结冰的路面、破碎的家具。
- 角色利用环境：踢沙迷眼、借墙反弹、用桌椅格挡。

## 感官轰炸（拒绝动作清单）
- **视觉**：骨节的模糊残影、飞溅的血沫、对手眼中的疯狂与恐惧。
- **听觉**：拳头砸肉的闷响、骨裂的脆响、粗重的喘息、倒地后的寂静。
- **触觉**：撕裂肌肉的灼烧感、掌心的沙砾、血液在指缝间的温滑。
- **嗅觉/味觉**：口腔中铜锈般的血腥味、恐惧产生的酸臭汗味、爆炸后的焦糊味。

## 让战斗有意义的三支柱
1. **赌注**：为保护他人而战 > 为自己而战。明确"输了的代价"。
2. **动机**：每个参战者必须有清晰、迫切的目标，不能"只是打起来"。
3. **后果**：战斗必须改变故事走向——留下的不仅是伤疤，还有心理创伤、愧疚、关系裂痕。

## 境界压制（修仙/玄幻专属）
- 高境界对低境界的碾压感：不是"更强"，而是"维度不同"。
- 低境界反抗高境界：不要靠蛮力，靠智慧、环境、规则漏洞、意志爆发。
- 越级战斗的代价必须巨大——主角不是无敌的。

## 禁止模式
- 禁止"回合制"战斗描写（你一拳我一拳）。
- 禁止战斗中角色进行超长独白或心理分析。
- 禁止战斗结果不影响后续剧情。
