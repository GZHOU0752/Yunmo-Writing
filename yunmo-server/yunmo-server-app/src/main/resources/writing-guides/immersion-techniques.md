# 代入感技法写作指南

## 一、自然信息交代

角色身份/外貌/背景通过**行动和对话**带出，禁止"资料卡式"直接罗列。

- ❌ "李明是一个二十三岁的程序员，身高一米七八，毕业于清华"
- ✅ "李明揉了揉酸胀的手腕，屏幕上的代码已经连续滚动六个小时了。组长路过时拍了拍他的肩：'清华出来的就是不一样。'"

## 二、画面代入法

开场先给画面（动作、环境、声音），再给信息。让读者"看到"而非"被告知"。

- 前 200 字必须有至少一个可拍成画面的具体动作或场景
- 禁止以世界观说明或角色介绍作为开篇

## 三、五感代入

场景描写中每 500 字加入 1-2 种五感细节：

| 感官 | 何时用 | 示例 |
|:---:|:---|:---|
| 视觉 | 定调（光线/颜色） | "月亮大得像路灯" |
| 听觉 | 紧张/寂静 | "楼上有人在拖椅子，很慢" |
| 嗅觉 | 危险/腐败/生机 | "烟里还有别的味——焦肉？" |
| 触觉 | 温度/材质 | "风灌进来，贴在后背的衬衫冰凉" |
| 体感 | 极端情绪 | "胃像被人攥了一把" |

## 四、欲望钩子

每章至少让读者产生一个"接下来会怎样"的好奇心：

- 不要直接回答本章提出的所有问题——留至少一个给下一章
- 揭示一个信息的同时埋下一个新疑问
- 章末钩子必须是具体画面，不是情绪总结

## 五、情绪用场景传达

- ❌ "她感到深深的孤独"  
- ✅ "她坐在空荡荡的食堂角落，筷子拨弄着米饭，窗外有对情侣走过"

- ❌ "他心中充满不甘"  
- ✅ "他把通知书折了又折，直到纸边起了毛，然后塞进裤兜最深处"

## 六、禁止的"代入感破坏"

- 角色忽然开始一段独白解释自己的动机（"你可能会问为什么..."）
- 旁白直接告诉读者"这是一个重要的决定"
- 连续三句以上心理分析（"他想...他意识到...他明白..."）
- 刚出场的新角色立刻被大段介绍背景——先让读者看到他的行为
