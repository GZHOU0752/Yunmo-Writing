# 六步人物心理分析写作指南

## 一、核心原则

每个重要角色在关键场景中的行为，必须经过以下六步推导。**禁止跳过步骤直接写行为。** 如果推导不出合理行为，说明前置铺垫不足，先补铺垫。

## 二、六步推导

### 第一步：当前处境
角色此刻面临什么局面？手上有什么牌？
- 他/她站在什么位置（物理+社会）？
- 周围有谁？谁在看他/她？
- 有什么资源可用？有什么限制？

### 第二步：核心动机
角色最想要什么？最害怕什么？
- 眼前这件事和他的长期目标有什么关系？
- 如果成功会得到什么？失败会失去什么？
- 此刻哪一个动机更强——贪婪、恐惧、责任、爱恨？

### 第三步：信息边界
角色知道什么？不知道什么？对局势有什么误判？
- 他知道哪些别人不知道的？
- 他不知道哪些读者知道的？（制造紧张）
- 他对自己处境的判断是对的还是错的？

### 第四步：性格过滤
同样的局面，这个性格会怎么反应？
- 冲动型：先动手再想
- 谨慎型：先观察再出手
- 阴险型：表面不动声色，暗中布局
- 果断型：快速判断，立刻执行
- **不要让一个角色的反应方式和其他角色雷同**

### 第五步：行为选择
基于以上四点，角色会做出什么选择？
- 这个选择是否独一无二？如果是另一个角色，会做不同选择吗？
- 选择必须有代价——每一次行动都应付出某种成本（体力/资源/关系/信息）
- 如果选择看起来"很合理但没味道"，换一个更有张力的方向

### 第六步：情绪外化
这个选择伴随什么情绪？用什么身体语言、表情、语气表达？
- ❌ "他感到愤怒" → ✅ "他指节发白，太阳穴突突直跳，半晌才从牙缝里挤出两个字"
- ❌ "她很紧张" → ✅ "她的手指绞着衣角，指节泛白"
- **禁止旁白替读者总结情绪。用具体身体反应替代抽象形容词。**

## 三、动笔前自检

每写完一个关键场景，自问：
1. 主角此刻利益最大化的选择是什么？如果他的选择和利益最大化不同，为什么？
2. 这场冲突是谁先动手，为什么非做不可？
3. 配角/反派是否有明确诉求、恐惧和反制？行为是否由"过往经历+当前利益+性格底色"驱动？
4. 反派当前掌握了哪些已知信息？哪些信息只有读者知道？有无信息越界？
5. 如果任何问题答不上来，先补逻辑链，再写正文。

## 四、信息越界判断

反派/配角不能基于他们不知道的信息行动：
- 主角的秘密计划 → 除非被监视/告密，否则反派不应知晓
- 主角的内心想法 → 除非有读心术设定，否则任何角色不应知晓
- 跨场景信息 → A地发生的秘密事件，B地的角色不应莫名其妙知道

**规则：每写一个角色的行动，反问"他是怎么知道该这么做的？"如果答不上来，就是信息越界。**
