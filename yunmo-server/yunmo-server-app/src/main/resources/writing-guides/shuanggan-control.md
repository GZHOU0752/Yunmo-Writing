# 爽感控制写作指南

## 一、读者留存公式

```
屈辱/危机（具体、当众、涉及脸或亲人）
    ↓ 尽量压缩，不拖戏
主角反击或获得筹码（信息/资源/能力/盟友）
    ↓ 要有代价或风险，不能白捡
阶段性爽点（打脸/升级/夺宝/认主）
    ↓
更大威胁露头（章末钩子）
```

## 二、怎么写屈辱（又不劝退）

- **具体**：谁、在哪、当着谁、损失什么（钱、名、命、亲人尊严）
- **短**：同一羞辱不重复三遍；读者恨完就进反击
- **留尊严**：主角可以怂，但要有一句硬话、一个眼神、一个藏牌让读者站他

## 三、怎么写反击与升级

| 类型 | 写法 | 追读点 |
|:---:|:---|:---|
| 打脸 | 当众、证据硬、对方失态 | 读者替主角出气 |
| 升级 | 数字/异象/他人震惊反应 | 可见成长 |
| 夺宝/认主 | 过程有险，不是伸手就有 | 紧张+收获 |
| 信息差 | 主角早知道，敌人不知道 | 期待揭晓瞬间 |

**禁止**：无铺垫碾压、反派集体降智、系统一次性讲完全部规则。

## 四、语言与场面

- 战斗/冲突：短句、动词、身体感（疼、喘、血、脱力）
- 少用形容词堆"恐怖如斯"，用旁人反应衬强度
- 对话可狂可冷，但要有性格；反派可以嚣张，台词要短

## 五、章末钩子（爽文向）

- 更强敌人露面、旧敌复返、升级副作用、亲人被挟、规则升级
- 最后一句话最好是挑衅、宣战或倒计时
