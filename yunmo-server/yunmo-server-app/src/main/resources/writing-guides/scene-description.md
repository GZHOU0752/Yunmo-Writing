# 场景描写指南

## 核心原则：五感入境法
读者进入一个新场景，应该像踏入一幅水墨画——先见远山、再观近水、最后聚焦到石上的一只栖鸟。由远及近、由外及内、由环境及人物。

## 三层描写法
1. **远景（一笔带过）**：山脉的轮廓、城池的剪影、天空的颜色——一个字"势"。
2. **中景（建立氛围）**：街道的宽窄、建筑的材料、人群的密度——一个字"气"。
3. **近景（角色交互）**：脚下的青石板裂缝、手边的茶杯花纹、对面人的微表情——一个字"触"。

## 五感填充（每次场景切换至少使用 3 种感官）
- **视觉**：不只是"看到了什么"，更是光线如何落下（斜阳/烛光/月色/萤火）。
- **听觉**：环境音（风声/雨声/市声/虫鸣/钟声），声音暗示空间的大小和材质。
- **嗅觉**：最直接唤起记忆和情绪的感官（雨后泥土味、焚香味、血腥味、药味、花香）。
- **触觉**：温度（寒/暖/湿/燥）、质感（粗糙/光滑/柔软/冰冷）、身体感受（疲惫/轻快/紧绷）。
- **味觉**：谨慎使用，仅在角色真的吃东西或尝到某物时提及。

## 场景即信息
- 场景不是背景板，场景本身在讲故事：破败的庙宇暗示曾经的辉煌，整齐的军营暗示纪律严明。
- 角色的环境反映其内心：孤独的角色身处空旷的房间，焦虑的角色在拥挤的街市。
- 场景切换暗示时间流逝：窗外的光从东斜到西沉，炉中的火从旺盛到余烬。

## 水墨画式描写法则
- **留白胜于填满**：三个精准的细节比十个模糊的概括更有力。
- **以一当十**：不说"繁华的都市"，写"街角的包子铺排了二十步长的队"。
- **动静相衬**：寂静中有一滴水的回声，喧闹中有一个人的沉默。

## 场景类型触发的描写密度
- **新地点首次登场**：必须三层描写（远景→中景→近景），建立空间感。
- **重返旧地**：只写变化的部分，暗示时间流逝或剧情影响。
- **动作场景中的场景**：环境描写融入动作——角色撞碎花瓶、滑过积水的路面、借柱子躲避。
- **对话场景中的场景**：环境作为"沉默的第三角色"——窗外的雷声配合争吵、落花伴随告白。

## 禁止模式
- 禁止开头大段静态描写后再进入剧情——描写应与角色行动同步推进。
- 禁止"此处风景如画""美不胜收"之类空洞概括——必须写具体细节。
- 禁止场景描写与剧情无关——描写的每个细节都应该暗示某种信息。
- 禁止场景间缺少过渡——除非有意制造跳跃感。
