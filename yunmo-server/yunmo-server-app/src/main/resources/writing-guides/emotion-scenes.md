# 情感场景写作指南

## 核心原则："藏"字诀
最高级的情感描写，是把情感藏起来让读者自己发现。不说"他很伤心"，写他"盯着茶杯里的涟漪，直到水凉透"。

## 感受 vs 情绪
- **情绪**：原始本能的生理反应（愤怒、恐惧、喜悦）——短暂但强烈。
- **感受**：经过认知加工的心理状态（憋屈、期待、怅然）——持久、产生回味。
- 网文要兼顾两者：用情绪的生理反应传递即时冲击，用感受的心理状态制造回味。

## 突破三大障碍（让读者感同身受）
1. **虚构障碍**：用具象化五感代替抽象词。不说"他很猥琐"，写"他咧嘴时舌尖舔过发黄的牙床"。
2. **损失归属障碍**：让读者感知"这事可能发生在我身上"——正义被践踏、公平被破坏、信任被辜负。
3. **情绪发出者障碍**：去掉对话前缀，动作直连情绪，省略主语让读者代入。

## 以景写情
- 喜→天晴、花开、风暖。
- 悲→雨冷、灯孤、叶落。
- 怒→雷暴、浪涌、火燃。
- 惧→雾重、影乱、声寂。
- 但不要机械对应。有时悲伤配暖阳更有反差力。

## 留白技法
- 最高浓度的情感不需要写出来。写角色"转身后，门关了很久，久到他以为她不会追出来"。
- 对话中的沉默比语言更有力：人物停顿、转移话题、答非所问，都是情感的裂缝。
- 省略高潮时刻的直接描写，只写前因和后果，让读者脑补最痛的部分。

## 情感的心理层次
- 表层反应：生理变化（心跳加速、手心出汗、瞳孔收缩）。
- 中层反应：行为变化（语速变化、肢体僵硬、逃避眼神）。
- 深层反应：认知变化（重新评估某人、质疑自己的信念、做出不可逆的决定）。

## 损失 > 收益
- 人类对损失的敏感度是对收益的 2-3 倍。
- 先制造低谷（角色失去重要之物），再超越预期（获得比失去更好的），强化爽感。
- 但不能滥用——每次损失都必须是剧情必须，不能为虐而虐。

## 禁止模式
- 禁止角色直白陈述内心（"我很生气因为你背叛了我"）。
- 禁止用抽象形容词代替具体描写（"他很温柔"——写他做了什么温柔的事）。
- 禁止情感变化不影响行为——角色内心翻江倒海但行为毫无变化是不真实的。
