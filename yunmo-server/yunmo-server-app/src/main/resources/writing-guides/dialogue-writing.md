# 对话写作指南

## 核心原则：对话即战斗
每个角色开口说话时都有目标——说服、威胁、试探、逃避、取悦、挑衅。真正的人很少直白说感受，更多是用话语进行攻防。

## 每种声音必须独特
- **词汇选择**：书生用典、武将粗犷、商贾圆滑、孩童天真。每个人的词汇库不同。
- **句式长度**：急性子短句连发、谋士长句迂回、上位者言简意赅。
- **口头禅与停顿习惯**：有人爱说"就是说"、有人习惯"嗯"开头、有人永远不直接回答问题。
- **身份决定语气**：对上级恭敬、对平级随意、对下级命令。同一人在不同对象面前语气不同。

## 潜台词：灵魂所在
- 人在生气时不会说"我很生气"，而是变得过分礼貌，或者用沉默施压。
- 受伤的人转移话题，害怕的人虚张声势，心虚的人主动攻击。
- 对话中最有趣的部分，是角色"真正想但没说出口"的话。
- 用动作暴露潜台词：嘴上说"没事"，手却把茶杯捏得泛白。

## 动作节拍穿插对话
- 对话不是电台剧。写角色说话时的身体语言：手指轻叩桌面、眼神飘向窗外、深吸一口气再开口。
- 动作可以打断对话、替换对话、反讽对话。嘴上说"我很好"的同时摔门而出。
- 长对话必须用场景动作分段：倒茶、起身、踱步、望向远方——让读者感受时间的流逝。

## 对话格式规则
- **换人换行**：每个新说话者另起一段，这是硬规则。
- **避免过度使用"某某说"**：用动作代替对话标签。与其写"他生气地说"，不如写"他—拳砸在桌上，'你再说一遍'".
- **沉默也是对话**：角色不回答、停顿、换话题，都是重要的"台词"。

## 对话的信息释放
- 对话是传递信息最自然的方式，但不要让角色说他们已经知道的东西（"如你所知……"是坏的）。
- 用冲突释放信息：在争吵中暴露真相，在审讯中泄露秘密，在酒后吐真言。
- 信息要有层次：先给表象，再给真相，最后给角色自己对真相的反应。

## 场景类型触发的对话密度
- **动作场景**：对话极简，以喊叫、命令、短促交流为主。
- **对峙场景**：对话密集，每句话都是武器，节奏紧凑。
- **日常场景**：对话轻松但必须有信息量，展现人物关系和性格。
- **情感场景**：对话与沉默交替，未说出口的比说出口的多。

## 禁止模式
- 禁止角色完美陈述内心独白（"我知道你在想什么，你认为我……"）。
- 禁止全员说话风格一致——五个角色读起来像同一个人。
- 禁止大段独白没有动作穿插——超过三句连续对话就应该插入动作。
- 禁止"如你所知"式信息灌输对话。
