# 章末钩子写作指南

## 一、商业价值

数据上，章末 50 字决定下一章打开率。  
钩子不是"悬念"两个字，是读者脑子里自动冒出的具体问题：「然后呢？」「他会不会死？」「那句话什么意思？」

## 二、合格钩子标准

| 标准 | 说明 |
|:---:|:---|
| 具体 | 人、物、话、数字、倒计时，可拍成画面 |
| 叠层 | 本章已有阶段性结果，钩子是结果之上的升级 |
| 可接 | 下一章开头能直接接这一屏，不需大段回忆 |
| 非空 | 不是"他感到不安""命运齿轮转动" |

## 三、五种钩子类型

### 1. 威胁升级
更强敌人露面、规则二段触发、追杀到场、倒计时启动
→ 读者心理：危险近了

### 2. 信息反转
证人反水、物证矛盾、身份露头、一句话推翻之前认知
→ 读者心理：我之前猜错了

### 3. 情感悬置
话没说完、误会加深、选择逼迫、情感告白被打断
→ 读者心理：关系要炸了

### 4. 资源诱惑
开门、宝箱、权限、邀约、金手指新能力
→ 读者心理：想看他拿不拿

### 5. 倒计时
天亮行刑、契约到期、毒发、修为崩溃限时
→ 读者心理：时间不够了

## 四、写法模板

**模板 A（台词）**
> 他转身要走。  
> 她忽然说："你口袋里的那张纸，是我父亲的。"

**模板 B（动作/画面）**
> 灯灭了。  
> 不是停电——灯罩上有指印，朝外。

**模板 C（规则/系统）**
> 系统提示音在脑子里响：【第二条规则已激活。】  
> 第一条他还没活下来。

## 五、自检

- [ ] 能否用一句话向朋友描述"为什么点下一章"？
- [ ] 本章主线是否有结果（赢/输/逃/谈崩）？
- [ ] 钩子是否依赖"角色突然变傻"？
