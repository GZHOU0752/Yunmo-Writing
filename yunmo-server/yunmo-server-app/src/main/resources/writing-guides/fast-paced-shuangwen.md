# 快节奏爽文写作指南

## 一、定位

**快节奏爽文** = 手机追读友好的"高压—释放—再高压"循环。  
每章 2000-3500 字，一章一个可见变化，三章一个小高潮。

目标：**读者划到章末时，手指已经想点"下一章"。**

## 二、章内时间表

| 区间 | 字数约 | 任务 |
|:---:|:---:|:---|
| 开场 | 0-200 | 接悬念/冲突，零背景课 |
| 加压 | 200-1200 | 危机具体化、主角受挫或发现缺口 |
| 释放 | 1200-2200 | 反击、收获、打脸、破局之一落地 |
| 抬杠 | 最后 200-400 | 钩子：新威胁/新信息/关系变 |

## 三、信息密度

删一切"拖划走"的句子。删掉后剧情、情绪、关系完全不变 → 删或改。

**允许**：
- 动作链
- 带信息的对话
- 功能/体感环境（各 ≤20 字）
- 极短内心（立刻接行动）

**禁止**：
- 世界观综述
- 连续三句心理分析
- 章末总结成长

## 四、呼吸段

高压后可以给 3-5 句呼吸，但呼吸里要**盘算下一步**：

❌ 弱：「他大口喘气，觉得很累。」  
✅ 强：「他喘匀了，拇指蹭过刀口——偏了三寸，下次先废右手。」

## 五、追读自检

- [ ] 前 200 字有冲突？
- [ ] 主角 500 字内主动？
- [ ] 本章局势/关系/资源至少变一处？
- [ ] 章末有具体钩子？
- [ ] 无"不是…而是…"、无破折号说明书？
- [ ] 无世界观综述、无连续三句心理分析？
