# 去AI味写作指南

## 一、高频句式替换

写完后逐段扫描，命中左列 → 按右列策略改：

| AI / 模板句 | 替换策略 | 示例方向 |
|:---|:---|:---|
| 不是 A，而是 B | 删对照，写 B 的动作 | "他咬牙，一掌拍下" |
| X——Y 解释 | 拆两句或删 Y | "门开了。冷风灌进来。" |
| 下一刻/紧接着 | 删，用动作衔接 | "他抬脚。地板吱呀。" |
| 与此同时 | 删，切场景用新段落+动作 | "院里有人喊。" |
| 他意识到… | 删，用半句念头+行动 | "（坏了。）他转身就跑。" |
| 这意味着… | 删，写后果 | "掌柜脸色白了。" |
| 显然/毫无疑问 | 删 | 直接写事实 |
| 心头一震/瞳孔骤缩 | 删，写具体反应 | "他手一抖，茶洒了。" |
| 空气仿佛凝固 | 删，写具体静 | "没人接话。钟响了一下。" |
| 非常/极其/十分 | 删或换具体细节 | "烫得他缩手" |
| 他感到愤怒/恐惧 | 删标签，写身体 | "他喉结滚了一下，没出声。" |
| 不禁/顿时/瞬间 | 删，直接写动作 | "他笑了"而非"他不禁笑了" |

## 二、段落级替换

| 问题段 | 改法 |
|:---|:---|
| 连续 3 句"他想" | 保留 1 句，其余改动作/对话 |
| 连续短句机械 | 合并 2 句或插入环境/对话打断 |
| 设定告示+内心复读 | 删告示，留一次碰壁后果 |
| 章末感悟 | 改具体钩子画面 |

## 三、禁止的"假钩子"

- "欲知后事如何，且听下回分解"
- 纯梦境/幻觉揭示
- 作者旁白预告"更大的阴谋在等待"
- 情绪总结："他知道，暴风雨即将来临"

## 四、自检清单

- [ ] 全文搜索"不是""而是""——""意识到""紧接着""与此同时""不禁""顿时""瞬间"
- [ ] 随机抽 5 句，能否一眼读完？
- [ ] 删掉所有"形容词堆叠"后剧情仍通？
- [ ] 抽读任意连续 200 字，是否像人话？
