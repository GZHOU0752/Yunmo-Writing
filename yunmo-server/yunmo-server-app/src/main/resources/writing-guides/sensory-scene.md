# 五感场景切入写作指南

## 一、核心原则

环境描写不为"美"，为**信息和情绪**。  
每次场景切换或新地点，至少调动 2 种感官。

## 二、感官层次

| 感官 | 何时用 | 示例 |
|:---:|:---|:---|
| 视觉 | 定调（光线/颜色/空间感） | "月亮大得像路灯，照得刀疤发白" |
| 听觉 | 制造紧张/寂静/远近 | "楼上有人在拖椅子，很慢" |
| 嗅觉 | 暗示危险/腐败/生机 | "烟里还有别的味——焦肉？" |
| 触觉 | 强化身体感、温度对比 | "风灌进来，贴在后背的衬衫冰凉" |
| 体感/内脏 | 极端情绪 | "胃像被人攥了一把" |

## 三、描写法则

- 环境描写 ≤ 20 字，紧跟动作，不单独空镜
- 一个具体细节 > 三个抽象形容词
- 用角色的感受过滤环境，不是客观描述
- 环境变化 = 局势变化信号（灯灭、风起、人声远去）

## 四、禁止

- 纯风景空镜（"夕阳西下，金色的余晖"）
- 百科式地点介绍（"这座城有三百年的历史"）
- 环境与情绪脱节（紧张场景写"窗外鸟语花香"除非是反差用意）
