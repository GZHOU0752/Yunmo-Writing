# 节奏人工化写作指南

## 一、AI 节奏病

- **太匀**：句长差不多、段长差不多，读感像节拍器
- **太断**："他走。他停。他看。"像机器人
- **太排比**：三段式、对称句，像演讲稿

读者会觉得"假"，追读下降。人工节奏 = **乱中有序**。

## 二、打碎均匀

1. **超过 40 字必拆**（紧张处 20 字就拆）
2. **连续 3 个短句**：合并其中 2 句，或中间插环境/对话
3. **同段开头重复**（他…他…他…）：换主语或换句式
4. **禁止**同一章内同一套路短语 >1 次（心下一沉、目光一凝等）

## 三、注入可控瑕疵

| 可用 | 慎用 |
|:---:|:---|
| 单句成段 | 全文单句成段 |
| 半句话、改口 | 整段口语废话 |
| 角色口癖 | 全员同一口癖 |
| 突然沉默 | 连续沉默无推进 |

原则：瑕疵服务**性格与紧张**，不服务凑字数。

## 四、紧张 vs 舒缓

| 状态 | 句式 |
|:---:|:---|
| 紧张/战斗/对峙 | 短、断、动词多 |
| 蓄势/布局 | 可略长，但信息密度要高 |
| 舒缓呼吸 | 3-5 句，但必须盘算下一步 |

## 五、追读节奏

- 章首 200 字：短+密+有事
- 章末 200 字：短+钩子，不要长抒情拖慢点击欲
- 中段每 800-1200 字节奏至少一次变化（静→动，或动→更动）

## 六、自检

把一章最后 500 字朗读：
- 若喘不过气 → 紧张段合格
- 若像念课文 → 继续拆句、加动作
- 若听不出谁说话 → 对话加区分度
