# 对话潜台词写作指南

## 一、潜台词与追读

读者追对话，常追的是**「他到底什么意思」**。  
表层聊 A，底层争 B——读者会忍不住往下划。

## 二、五种技法

| 技法 | 写法 | 效果 |
|:---:|:---|:---|
| 回避 | 问东答西、突然改话题 | 有鬼 |
| 试探 | "昨晚去哪了"→"加班"→"哪家加班到血腥味" | 压迫 |
| 威胁包装 | "保重"+ 关门声重 | 不寒而栗 |
| 沉默 | 该答不答，转喝水、看窗 | 比骂狠 |
| 双关 | 一句可作两种理解 | 揭晓时爽 |

## 三、权力差怎么写

- 打断、抢话、故意叫错名字、笑而不答
- 弱势方：短句、含糊；强势方：问句多、停顿少
- **勿用旁白**解释"他在暗示"

## 四、节奏

- 关键信息延迟半拍给出
- 每 4-6 句对话插动作/环境
- 对话结束必须改变关系或局势（哪怕只是一寸：更冷、更惧、更贪）

## 五、无效判定

删掉对话剧情完全不变 → 重写，加入隐瞒/试探/代价。

## 六、自检

- [ ] 表层与底层是否不同？
- [ ] 是否有试探或隐瞒？
- [ ] 对话后局势是否变化？
