# 短句网感文风写作指南

## 一、目标读者怎么读你的书

- 竖屏、碎片化、随时会被消息打断
- 他们不是在"欣赏文学"，是在**找下一口刺激**
- 句子越长，划走概率越高；段落越长，越像作业

把文风压到：短、快、有网感、有信息。

## 二、句子级规则

### 长度
- 默认：一句话不超过一行屏（约 20-35 字）
- 紧张处：5-15 字一句，可单句成段
- 超过 40 字：强制拆成两句，或删修饰

### 形容词纪律
- 删掉"非常、极其、十分、无比、瞬间、不禁、顿时"连击
- 一个具体细节 > 三个抽象形容词

### 网感 ≠ 烂梗
- 允许：口语、吐槽、半句话、角色口癖
- 禁止：弹幕式"？？？"刷屏、括号表情包、脱离剧情的热梗堆砌

## 三、段落级规则

- 叙述段 1-3 行；对话单独成行
- 每段只做一件事：打一个动作 / 说一句狠话 / 给一个细节
- 800-1200 字内至少一次"局势变化"（打脸、收获、反转、关系变）

## 四、描写：只为"划下去"服务

| ✅ 要 | ❌ 不要 |
|:---:|:---|
| 月亮大得像路灯，照得刀疤发白 | 月色很美，银辉洒落 |
| 泡面坨了，油膜凝一层 | 食物已经失去美味 |
| 他骂了一句，转身就走 | 他非常愤怒地离开 |

## 五、追读自检

- [ ] 随机抽 5 句，能否一眼读完？
- [ ] 删掉所有形容词堆叠后剧情仍通？
- [ ] 本章有没有让读者想吐槽/想骂/想笑的一句人话？
- [ ] 章末是画面不是感想？
