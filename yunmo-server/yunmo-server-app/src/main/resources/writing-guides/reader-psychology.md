# 读者心理学与追读力写作指南

## 一、期待管理

- 在读者期待释放时，适当延迟以增强快感；在读者即将失去耐心时，立即给反馈
- 延迟不是水字数——是用新信息/新冲突/新视角来"拖"，让读者觉得等待值得
- **规则**：连续两章无爽点兑现 → 第三张必须给，否则读者会弃

## 二、信息落差

- **让读者比角色多知道一点**：制造紧张感——"快跑啊！你不知道后面有人！"
- **让读者比角色少知道一点**：制造好奇心——"他藏了什么在背后？"
- 动态切换：紧张场景用"读者多知"，悬疑场景用"角色多知"
- **禁止**：角色突然知道不该知道的事来推动剧情（信息越界）

## 三、情绪节拍

```
压制 → 释放 → 更大的压制 → 更大的释放
```

- 释放时要超过读者心理预期——给 80 分期待，还 120 分爽感
- 压制不能连续超过 2 章不给任何释放
- 每一段压制后，释放的规模应该比上一次更大

## 四、锚定效应

- 先给读者一个参照：对手有多强、困难有多大
- 然后再展示主角的表现——对比越强烈，爽感越强
- **规则**：越级挑战前必须展示对手的实力（战绩/他人评价/主角吃亏），不能直接写"他一掌拍飞了元婴高手"

## 五、沉没成本

- 读者已投入的阅读时间是留存的关键
- 每章都要给出"继续读下去的理由"——钩子/悬念/爽点兑现/新信息
- **规则**：章末必须有至少一个具体未决点——不是情绪总结，是能拍成画面的钩子

## 六、代入感维护

- 主角的困境必须有普遍性（被欺压/不公待遇/被低估）——让读者觉得"这也是我"
- 主角的选择必须让读者觉得"我也会这么做"——至少"我能理解"
- **禁止**：主角通过读者无法理解的逻辑解决问题（"主角光环"）
- **共鸣锚点**：主角每章至少一个能让读者产生共鸣的瞬间（愤怒/心疼/得意/紧张）

## 七、信息分层植入

- **基础信息**：在行动中自然带出，不超过 2 句
- **关键设定**：结合剧情节点揭示，每次不超过 3 句
- **严禁**：大段灌输世界观——超过 5 句纯说明即判定为"百科式灌设定"
- **规则**：每章新设定 ≤ 2 条，旧设定至少应呼一次
